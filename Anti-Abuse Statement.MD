# 反滥用声明（Anti-Abuse Statement）

## 1. 声明目的

本项目旨在提供对 Oracle Cloud Infrastructure（OCI）资源的合法管理、自动化运维及技术研究能力，仅供学习、研究、测试及符合相关法律法规和服务条款的场景使用。

项目作者不鼓励、不支持、不协助任何形式的滥用行为。

---

## 2. 合规使用要求

用户在使用本项目时，应严格遵守：

* 所在国家或地区的法律法规；
* Oracle Cloud Infrastructure（OCI）服务条款；
* Oracle Cloud Infrastructure（OCI）可接受使用政策（Acceptable Use Policy）；
* 相关第三方服务协议；
* 网络安全及数据保护相关法律法规。

用户应自行确认其使用行为的合法性与合规性。

---

## 3. 禁止用途

严禁将本项目用于包括但不限于以下行为：

### 账号相关滥用

* 批量注册云服务账号；
* 规避实名认证要求；
* 绕过账号限制机制；
* 规避资源配额限制；
* 利用漏洞获取额外资源；
* 非授权账号管理；
* 账号共享或交易；

### 资源滥用

* 恶意占用免费资源；
* 批量抢占资源；
* 规避资源回收机制；
* 规避官方资源管理策略；
* 超出服务商允许范围的自动化行为；
* 利用自动化手段影响平台正常运行；

### 网络与安全相关行为

* 网络攻击；
* DDoS 攻击；
* 扫描、爆破或渗透未授权目标；
* 恶意代理服务；
* 垃圾邮件发送；
* 恶意流量转发；
* 挖矿活动；
* 恶意软件传播；
* 数据窃取或非法采集；

### 违法违规行为

* 侵犯他人合法权益；
* 违反知识产权规定；
* 违反数据保护法规；
* 从事任何违法犯罪活动。

---

## 4. 关于免费资源的特别声明

本项目可能包含与 OCI 免费资源管理相关的功能。

项目作者不承诺：

* 免费资源长期可用；
* 免费套餐政策保持不变；
* 免费实例能够永久保留；
* 资源不会被系统回收；
* 账号不会受到限制；
* 账号不会被暂停或终止。

OCI 的资源政策、风控策略及服务规则可能随时调整，最终解释权归 Oracle 官方所有。

本项目提供的自动化管理能力仅用于合法资源管理场景，不应被理解为规避 Oracle Cloud Infrastructure 官方资源回收机制、风控策略、配额限制或服务政策的工具。项目作者不保证任何资源保活、实例续存或免费资源长期可用性。用户因相关功能使用导致的资源回收、账号限制、服务终止或费用产生，应自行承担全部责任。
